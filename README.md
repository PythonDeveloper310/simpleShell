# simpleShell
simpleShell is a minimal shell made in C for learning (me learning).
This is only for Windows, if you want to run on linux you can fork this repo, modify some code and then compile.

```
user@simple-shell:~/ > help
Available Commands:
  help - Show this help message.
  echo <msg> - Display the words in <msg> to the screen.
  add <num1> <num2> - Adds num1 with num2 and display the result.
  clear - Clears the screen.
  version/ver - Display the version.
  about - Show simpleShell information.
  ezfetch - An easy and simple version of neofetch/fastfetch.
  exit - Exits shell.
```
